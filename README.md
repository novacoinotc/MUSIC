# SYNTH FORGE

AI-Powered Melodic Techno Generator - Create unique techno tracks with real-time synthesis.

## Features

- **Kick Designer** - Shape your kick with punch, sub, decay, pitch and drive controls
- **Bass Generator** - Subtractive synthesis with filter, envelope, and glide controls
- **Melody Generator** - AI-generated melodies with scale selection (Minor, Phrygian, Harmonic Minor, Dorian, Locrian)
- **Hi-Hat Module** - Adjustable patterns (Straight, Offbeat, Shuffle, Complex)
- **Pad Synth** - Atmospheric pads with multiple chord voicings
- **Structure Builder** - Build complete tracks with Intro, Build, Drop, Breakdown, and Outro sections
- **Real-time Visualizer** - See your waveforms as you create
- **Export** - Download your tracks as audio files

## Tech Stack

- **Next.js 16** - React framework
- **Tone.js** - Web Audio synthesis
- **TailwindCSS** - Styling
- **Zustand** - State management
- **TypeScript** - Type safety

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## How to Use

1. **Click Play** to start the audio engine and hear your track
2. **Adjust the knobs** on each module to shape your sound
3. **Use the Structure Builder** to arrange your track sections
4. **Click Randomize All** for instant inspiration
5. **Click Export** to download your creation
